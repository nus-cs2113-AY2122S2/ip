| AVAILABLE COMMANDS |
  1. list
        Displays the full list of tasks that have been saved.
  2. todo <task name>
        Creates a new todo list entry.
  3. event <event name> /at <time range>
        Creates a new event entry with its accompanying time of occurrence.
  4. deadline <deadline name> /by <due date>
        Creates a new deadline entry with its accompanying due date.
  5. mark <task entry number>
        Marks the entry as done, see list for entry numbers.
  6. unmark <task entry number>
        Marks the entry as incomplete, see list for entry numbers.
  7. help / ?
        Displays this help menu.
  8. bye
        Displays the farewell message and closes the application.
